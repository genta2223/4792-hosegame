## 📁 ディレクトリ構成
- **/docs**: GitHub Pages 公開用フォルダ
- **/public_html**: ロリポップ（宣伝LP）アップロード用フォルダ

詳細は [walkthrough.md] をご確認ください。

## 🎮 ブラウザで遊ぶ
GitHub Pagesに公開されたURLにアクセスすることで、ブラウザで直接プレイできます。
(GitHub Pagesの設定で `/docs` フォルダを指定してください)

### PCでの操作
- **十字キー**: 移動 / 選択
- **Z / スペース**: Aボタン（決定 / ダッシュ）
- **X**: Bボタン（キャンセル / ステータス確認）

### スマホでの操作
画面上の**バーチャルパッド**を使用して操作してください。
- 画面が横向き（ランドスケープ）の方が快適にプレイできます。
- 音が出ない場合は、画面を一度タップしてください。
